### icons-platform

#### 用于图标服务平台

文件夹介绍

```
├── .cache   //临时文件存储
├── bin    //启动文件
├── config  //项目配置
├── contents_images //图片存储
├── contents_svg //svg存储
├── controllers  //业务实现
├── core  //核心代码入数据操作
├── docs  //仓库文档
├── logs //服务日志
├── node_modules //项目依赖
├── public  //静态资源
├── routes //路由器
├── sql //数据库结构备份
└── views //视图层
```
