第一步：使用font-face声明字体 
@font-face {font-family: 'pf-icons';
  src: url('pf-icons.eot'); /* IE9*/
  src: url('pf-icons.eot?#iefix') format('embedded-opentype'), /* IE6-IE8 */
  url('pf-icons.woff2') format('woff2'),/* chrome、firefox, the latest version*/
  url('pf-icons.woff') format('woff'), /* chrome、firefox */
  url('pf-icons.ttf') format('truetype'), /* chrome、firefox、opera、Safari, Android, iOS 4.2+*/
  url('pf-icons.svg#iconfont') format('svg'); /* iOS 4.1- */
}
第二步：定义使用iconfont的样式(use iconfont.css) 
/*
 * 1、在支持css3游览器。建议使用，eg: <i class="i-eb01"></i>
 * 2、在支持伪类:before,:after游览器下。建议使用，eg:<i class="platform-icons  i-eb01"></i>
 * 3、低版本游览器模式（如：IE6,IE7,IE8...）。建议使用，eg:<i class="platform-icons ">&#xeb01</i>
 */
/* 兼容所有版本写法 */
.platform-icons {
    font-family:"pf-icons" !important;
    font-size:16px;
    font-style:normal;
    -webkit-font-smoothing: antialiased;
    -webkit-text-stroke-width: 0.2px;
    -moz-osx-font-smoothing: grayscale;
}
[class*="i-"]:before {
    font-family:"pf-icons" !important;
    font-style:normal;
    -webkit-font-smoothing: antialiased;
    -webkit-text-stroke-width: 0.2px;
    -moz-osx-font-smoothing: grayscale;
}
第三步：挑选相应图标并获取字体编码，应用于页面 
//支持css3
<i class="i-eb01"></i>

//支持IE9+
<i class="platform-icons i-eb01"></i>

//支持IE6,7,8+
<i class="platform-icons">&#xeb01</i>
